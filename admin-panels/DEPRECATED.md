# Deprecated — use Chatwoot Settings instead

Static pages in this folder are **no longer the supported admin UI**.

Telephony (IVR, routing) and future BlinkOne admin features are integrated in the Chatwoot dashboard:

- **Settings → IVR flows**
- **Settings → Call routing (ACD)**

See [docs/blinkone/CHATWOOT_TELEPHONY_ADMIN.md](../docs/blinkone/CHATWOOT_TELEPHONY_ADMIN.md).

`/admin` on nginx now redirects to the Chatwoot app.
